## 1. Overview

This document defines the official color palette used in Abyss Forge.
The palette is inspired by the visual style of Neo Geo / SNK arcade games, emphasizing strong contrast, saturated colors, and intentional color limitations.

The goal is to ensure:

Visual consistency across all assets

Readability on small screens (mobile-first)

A classic 1990s arcade aesthetic

This palette is a deliberate artistic choice, not a technical limitation.

## 2. Palette Specifications

Total colors: 32

Color format: HEX

Palette type: Fixed, project-wide

Primary tool: Piskel (custom palette)

All visual assets must use only the colors defined in this document.

## 3. Usage Rules
### 3.1 Color Limits per Asset

Character sprites:

Maximum 16 colors per sprite

Enemies and NPCs:

Maximum 16 colors per sprite

Backgrounds & environments:

May use up to 32 colors

UI elements, icons, FX:

Must use colors from this palette

No external colors or smooth gradients allowed
